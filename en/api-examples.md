## About the Dofbot

The Dofbot is a 6DOF robotic arm used for the sorting phase of the challenge.

### Features
- 6 degrees of freedom
- Positioning precision
- ROS2 compatible

### Usage
- Automatic waste sorting
- Object manipulation
