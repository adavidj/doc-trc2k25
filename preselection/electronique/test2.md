# Test 2 : La boîte noire

Ce test porte sur l'analyse d'un circuit inconnu (boîte noire).

## Objectifs

- Identifier les composants internes
- Mesurer les signaux
- Déduire le fonctionnement

## Méthodologie

- Inspection visuelle
- Tests de continuité
- Analyse des signaux
