# Test Final : Système de convoyeur

Ce test final intègre tous les concepts appris pour créer un système de convoyeur robotique.

## Objectifs

- Intégrer ROS2, POO et pathfinding
- Contrôler un convoyeur automatisé
- Gérer la communication entre composants

## Architecture

- Nœuds ROS2 pour chaque composant
- Classes Python pour la logique
- Interface utilisateur
